# 개별연구 신청 시스템 웹 와이어프레임

이 폴더는 「개별연구 신청 시스템 페이지 이동 구조 확정안」을 기준으로 만든 기능 검증용 웹 와이어프레임만 보관합니다.

## 파일 구성

- `wireframe-system.html`: 전체 사용자 흐름, 학생 화면, 교직원 화면, 상태/모달, 컴포넌트 보드
- `styles/wireframes.css`: 와이어프레임 전용 흑백/회색 스타일
- `scripts/wireframes.js`: 화면 간 이동 버튼 연결

## 포함 범위

- L1 로그인
- S1~S10 학생 화면
- A1~A4 교직원 화면
- 주요 상태 및 모달
- 반복 사용 컴포넌트
